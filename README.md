# Superstore Sales & Profitability Analysis Dashboard

## Project Overview

This project analyzes Superstore sales performance, profitability trends, customer behavior, and discount impact using SQL and Power BI. The goal was to identify key business insights and uncover the root causes of profit loss.

## 📊Tools Used

SQL (PostgreSQL)

Power BI

DAX

Excel

## 📈Key Analyses Performed




Sales trend analysis

Profit analysis

Loss analysis

Sub Category analysis

Customer profitability analysis

Discount impact analysis

Shipping mode analysis

## 🔍 Key Insights

>November recorded the highest sales volume due to increased order quantities; however, profitability did not increase proportionally because of High discounts.

>High discounts above 50% frequently resulted in negative profits.

>High sales did not always translate into high profit, particularly for Tables and Bookcases due to 50%-60% discounts

>Some high-revenue customers generated negative profits because of bulk discounted purchases.

>Standard Class shipping generated the highest sales and profit.

## 📌 Business Recommendations

### Reduce aggressive discounts on low-margin products.

### Monitor loss-making customers and bulk discount orders.

### Improve profitability checks before approving large discounts.
